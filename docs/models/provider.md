# Provider

provider is the integration provider name (e.g., "stripe", "razorpay")

## Example Usage

```python
from flexprice.models import Provider
value: Provider = "stripe"
```


## Values

| Name       | Value      |
| ---------- | ---------- |
| `STRIPE`   | stripe     |
| `RAZORPAY` | razorpay   |
| `PAYPAL`   | paypal     |
